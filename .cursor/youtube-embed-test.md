# בדיקת הטמעת יוטיוב (לא מ-localhost)

יוטיוב לא מאפשר הטמעה מדומיין `localhost` / `127.0.0.1`. כדי לבדוק שהסרטונים מתנגנים:

## אופציה 1: ngrok (פשוט)

1. הורדי מ־https://ngrok.com (או `winget install ngrok`)
2. הרצתי את שרת האתר כרגיל (למשל על פורט 5000)
3. בטרמינל נפרד: `ngrok http 5000`
4. ngrok ייתן לך כתובת כמו `https://abc123.ngrok.io` – פתחי אותה בדפדפן
5. היכנסי לשיר עם סרטון ולחצי "השמע/הסתר וידאו" – הסרטון אמור להתנגן

## אופציה 2: localhost.run (בלי התקנה)

1. הרצתי את השרת על פורט 5000
2. בטרמינל: `ssh -R 80:localhost:5000 localhost.run`
3. תשתמשי בכתובת שהפקודה מציגה (משתנה בכל הרצה)

## סיכום

- **ב-localhost:** "הסרטון לא זמין" – צפוי, יוטיוב חוסם.
- **בכתובת ציבורית (ngrok / פרודקשן):** הסרטונים אמורים להתנגן עם הקוד הנוכחי.
